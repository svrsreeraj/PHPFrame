Easily interact with the Authorize.Net AIM API.

Example Authorize and Capture Transaction against the Sandbox:

  1.  <?php require_once 'AuthorizeNet.php'
  2.  $sale = new AuthorizeNetAIM;
  3.  $sale->setFields(
  4.      array(
  5.     'amount' => '4.99',
  6.     'card_num' => '411111111111111',
  7.     'exp_date' => '0515'
  8.     )
  9.  );
  10.  $response = $sale->authorizeAndCapture();
  11.  if ($response->approved) {
  12.      echo "Sale successful!"; } else {
  13.      echo $response->error_message;
  14.  }
  15.  ?>

Note: To send requests to the live gateway, either define this: define("AUTHORIZENET_SANDBOX", false); -- OR -- $sale = new AuthorizeNetAIM; $sale->setSandbox(false);
